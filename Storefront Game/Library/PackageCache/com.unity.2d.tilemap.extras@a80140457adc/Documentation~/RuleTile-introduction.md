# Rule tiles and auto tiles

To change which tile Unity paints based on its surrounding tiles, use either of the following approaches:

- [Rule tiles](RuleTile.md), where Unity selects a sprite based on rules about each neighboring tile.
- [Auto tiles](AutoTile.md), where Unity selects a sprite based on floor layouts you define.
