# Install the 2D Tilemap Extras package

To install the 2D Tilemap Extras package, follow these steps:

1. From the main menu, select **Window** &gt; **Package Manager** to open the **Package Manager** window.
2. In the **Sources** section, select **Unity Registry**.
3. Select **2D Tilemap Extras** from the list of packages.
4. Select **Install**.
